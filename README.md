# raised-cosine-wavelet-shrinkage
This repository contains the R codes used to reproduce the simulation study presented in the manuscript: Reina, J. M., & Sousa, A. R. S. (2026). *Wavelet shrinkage based on the raised cosine prior*. **Journal of Statistical Computation and Simulation**. https://doi.org/10.1080/00949655.2026.2634886
